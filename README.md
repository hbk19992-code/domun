# 두문자 카드

법학 시험 준비를 위한 두문자 플래시카드 앱입니다.

## 기능

- **AI 추출**: 교재 PDF를 올리면 Gemini AI가 두문자 카드를 자동으로 추출
- **학습 모드**: 대분류/과목/단원별 필터링, 셔플 기능
- **기록형 분리 학습**: 일반 카드와 민사기록형 카드를 분리하고, 기록형만 답안 자동채점 적용
- **음성 학습**: 질문을 읽은 뒤 자동으로 카드를 뒤집고 정답까지 이어서 재생
- **추출 누락 점검**: 번호가 있는 원문은 추출 결과의 `sourceNumber`와 비교하고, 누락 의심 번호만 다시 보충 추출
- **카드 관리**: JSON 가져오기/내보내기, 카드 삭제, 대분류/과목/단원 일괄수정
- **Xteink X4 내보내기**: 대분류/과목/단원별 UTF-8 TXT 및 EPUB 생성

## 사용법

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 무료 Gemini API 키 발급
2. "AI 추출" 탭에서 키 입력 후 교재 PDF 업로드
3. 추출된 카드 확인 후 "내 카드에 추가"
4. "학습" 탭에서 공부 시작
5. Xteink X4에서 읽으려면 "관리" 탭의 "Xteink X4용 내보내기"에서 대분류/과목/단원을 선택한 뒤 TXT 또는 EPUB 파일 생성 후 X4 microSD 카드에 복사

## 카드 분류

카드는 `대분류 > 과목 > 단원` 순서로 정리합니다.
기존 Firestore 카드처럼 대분류 값이 없는 데이터는 앱에서 자동으로 `기본` 대분류로 표시됩니다.
관리 탭의 일괄 변경 기능으로 기존 카드 묶음을 원하는 대분류로 옮길 수 있습니다.


## Xteink X4 파일 내보내기

"관리" 탭의 "Xteink X4용 내보내기"에서 대분류, 과목, 단원을 선택한 뒤 파일을 생성합니다.

- `전체 / 전체 / 전체`: 모든 카드
- `대분류 / 전체 / 전체`: 특정 대분류 전체
- `대분류 / 과목 / 전체`: 특정 대분류의 특정 과목 전체
- `대분류 / 과목 / 단원`: 특정 대분류, 과목의 특정 단원

생성된 TXT 또는 EPUB 파일을 X4 microSD 카드에 복사해서 기본 리더로 열면 됩니다.

## 기술 스택

- React 18
- Vite
- Google Gemini API (gemini-2.5-flash)
- Firebase Firestore (카드 데이터 저장)
- localStorage (오프라인 캐시 및 임시 복구)

## 저작권 안내

이 앱은 사용자가 직접 카드를 만들어 사용하는 도구입니다.
기본 카드 데이터는 포함되어 있지 않습니다.
공개 배포본에는 저작권이 있는 강의자료·교재 기반 카드 데이터를 코드에 탑재하지 않습니다.
